# BiteUp Project Rules

- **Typography & UI**: BiteUp uses Blue as its primary color (#3b82f6).
- **Branding**: The app name is "BiteUp". No dots before or after the name.
- **Labels to Avoid**: Never add decorative/technical labels like "Live Feed", "Culinary Archive", "Archive [ID]", "Expert Mode", or similar metadata descriptions. Keep the UI focused strictly on the culinary content.
- **Authentication**: Use Firebase Authentication with Google Sign-In for persistence.
